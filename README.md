# Elden Ring Blog

El siguiente proyecto es el desarrollo frontend de un blog de Elden Ring en una aplicacion web.

## Features

- Ver el listado de posts
- Ver el contenido de los posts
- Buscar publicaciones por id
- Responsive


## Prerequisites

- [React](https://reactjs.org/)
